# FinG

FinG is a microfinance field collection and recovery management platform.

## Planned Architecture

- **Frontend Web:** Next.js
- **Mobile App:** Flutter
- **Backend API:** Spring Boot
- **Database:** PostgreSQL
- **Cache/Queue:** Redis
