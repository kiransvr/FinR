What it does

  • Loads portfolio data and checks required columns.
  • Computes arrear, collateral, and delinquency features.
  • Predicts which currently regular accounts are likely to default later.
  • Generates collections outputs for officers and supervisors.

Key outputs

  • Scored accounts ranked by risk
  • Top risky accounts for immediate attention
  • Daily visit plan for officers
  • Officer KPI summaries and feedback template

Workflow

1

Upload data

Loan officers upload a portfolio file from the dashboard.

2

Validate fields

The system checks whether the required columns are present.

3

Run scoring

Rules and ML create future default risk outputs.

4

Act on results

Officers use visit plans and KPIs to prioritize collections.
