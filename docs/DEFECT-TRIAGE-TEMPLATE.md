## Severity

Choose one:

- `Sev-1 Critical`: production blocker, data corruption, security breach, or complete outage.
- `Sev-2 High`: major feature unavailable with no reasonable workaround.
- `Sev-3 Medium`: partial feature impact with workaround available.
- `Sev-4 Low`: minor UX/content/cosmetic issue.

## Priority

Choose one:

- `P0`: fix immediately before any release.
- `P1`: fix in current sprint.
- `P2`: schedule for next sprint.
- `P3`: backlog candidate.
